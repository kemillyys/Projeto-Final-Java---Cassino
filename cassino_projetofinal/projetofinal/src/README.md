Organizacao do sistema:
- App.java: main, com menu inicial, login e menu do jogador
- Pessoa.java: classe base
- Jogador.java: herda de Pessoa
- Carteira.java: guarda o saldo do jogador
- Jogo.java: classe abstrata
- Roleta.java: herda de Jogo
- CacaNiquel.java: herda de Jogo
- Aposta.java: registra uma jogada
- Cassino.java: controla listas, cadastro, consulta, relatorio e arquivos

Menus:
- Menu inicial: fazer login, cadastrar usuario, ver usuarios cadastrados ou encerrar
- Menu do jogador: aparece depois do login e permite depositar saldo, jogar e consultar relatorios
